# 💰 Mario, o Educador Financeiro

Mario é um assistente inteligente focado em educação financeira, criado para ajudar pessoas a entender melhor seu dinheiro, organizar suas finanças e tomar decisões mais conscientes.

## 🚀 Objetivo

Desenvolver uma solução prática e acessível que:

- Explique conceitos financeiros de forma simples
- Ajude no controle de gastos
- Incentive o planejamento financeiro
- Apoie decisões mais inteligentes com dinheiro

## 🧠 Sobre o Projeto

Este projeto foi desenvolvido como parte de um laboratório prático, evoluindo de um template inicial para uma solução funcional com foco em:

- Experiência do usuário
- Clareza nas respostas
- Aplicação real no dia a dia financeiro

## ⚙️ Funcionalidades

- 📊 Orientação sobre orçamento pessoal  
- 💡 Dicas de economia e organização financeira  
- 📈 Introdução a investimentos  
- 🧾 Apoio no controle de despesas  

## 🛠️ Tecnologias Utilizadas

- Inteligência Artificial (IA generativa)
- Markdown para documentação
- Git & GitHub para versionamento
